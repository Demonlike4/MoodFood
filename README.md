# MoodFood – Selector de comida según estado de ánimo

## Descripción
Proyecto Integrado para el Grado de Desarrollo de Aplicaciones Web. Una plataforma que ayuda a combatir la fatiga de decisión sugiriendo alimentos basados en el estado emocional, uniendo nutrición y bienestar mental.

## Funcionalidades Principales
- Selección de estado de ánimo (UI React).
- API de sugerencias nutricionales (Symfony).
- Base de datos de recetas terapéuticas (MySQL).
- Explicaciones de psiconutrición integradas.

## Tecnologías
- **Frontend:** React.js
- **Backend:** Symfony Framework
- **Base de Datos:** MySQL
- **Estilos:** Tailwind CSS / Bootstrap
